# pyquack

A simple Python sync/async duckduckgo API wrapper.

## Installation

By default, pyquack will be synchronous and will use the [`requests`](https://requests.readthedocs.io) libary.
```
pip install -U pyquack
```

For async support,
```
pip install -U pyquack[async]
```
